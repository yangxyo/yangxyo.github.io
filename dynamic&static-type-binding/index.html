类型的动态绑定和静态绑定

发表于 2020-08-14

JavaScript 是动态绑定的语言，而 TypeScript 包装其成为静态绑定的语言。而什么是动态绑定，什么是静态绑定，它们的优劣又是什么呢？TypeScript 又为什么要“多此一举”？

绑定

绑定是一种关联（associate），也可以是一种分配（assign）。类型绑定是将声明的变量关联到特定类型的过程。

动态绑定就是，在程序运行的时候，将变量绑定到类型。
静态绑定，在编译时，编译器将变量绑定到类型。

编译与解释
