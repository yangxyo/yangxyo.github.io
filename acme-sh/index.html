用 acme.sh 从 Let's Encrypt 免费获取持续的 SSL 证书支持

发表于 2018-10-11 更新于 2021-08-22

acme.sh 简介

acme.sh 简化了 Let’s Encrypt 官方的证书签发过程。
目前主要有如下两类 Let’s Encrypt 证书的签发方式：

  • http 认证方式
  • dns 方式

http 需要 80 端口的贯通。
dns 方式不适用于所有的解析商，对域名的管理需要较高的权限。
